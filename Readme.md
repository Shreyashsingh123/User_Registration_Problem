# 🚀 User Registration Problem

## 📌 Introduction

# The **User Registration Problem** is a Python-based application designed to validate user details using **Regular Expressions (Regex)**.

# It ensures that all user inputs follow proper formats before acceptance.

## ✨ Features

# - ✅ Validate **First Name**

# - ✅ Validate **Last Name**

# - ✅ Validate **Email Address**

# - ✅ Validate **Phone Number**

# - ✅ Validate **Password**

## 🧪 Validation Rules (Example)

# - **FirstName** → Starts with capital letter, minimum 3 characters
# - **LastName** → Starts with capital letter, minimum 3 characters

# - **Email** → Standard email format (e.g., [abc@gmail.com](mailto:abc@gmail.com))

# - **Phone Number** → Country code +(space)+ number

# - **Password** →

# - Minimum 8 characters

      - At least 1 uppercase letter

      - At least 1 lowercase letter

      - At least 1 digit

      - Exactly 1 special character
